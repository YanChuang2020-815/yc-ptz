### 环境搭建
1.本项目串口通信开发基于RXTX，所以需要引用一些RXTX的依赖包：

- 拷贝 RXTXcomm.jar 到 JAVA_HOME\jre\lib\ext目录中；
- 拷贝 rxtxSerial.dll 到 JAVA_HOME\jre\bin目录中；
- 拷贝 rxtxParallel.dll 到 JAVA_HOME\jre\bin目录中；

2.安装USB驱动程序包
